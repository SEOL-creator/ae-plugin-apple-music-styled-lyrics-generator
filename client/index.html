How to use

  1. Create a composition for lyrics

  2. Add a text layer with lyrics

  3. Set text style

    Recommended style for 1080px height:

    Font: Apple SD Gothic Neo (for korean text)

    Weight: Bold

    Font Size: 60px

    Line Height: 72px

  4. Select the composition and hit 'Make' button

You can find a documentation with detailed explanation and usage in the Github repository.
